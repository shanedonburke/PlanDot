Creating items

To create an item, click on the New item button found in the toolbar or in the .

You may set the item's title (required), description, and location. The Description field also accepts Markdown formatting!

Specifying a date will cause the item to appear in the and .

You can create reoccuring items as well. When creating an item that occurs on a daily or weekly basis, you can select the days of the week on which it should occur. Selected days are highlighted in gray.
