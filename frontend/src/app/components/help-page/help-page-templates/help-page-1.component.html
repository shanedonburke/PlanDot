The group view


Groups are a way to organize your items. A group can contain many items, and one item may belong to multiple groups.

To create a group, go to the Group view by clicking the first button in the sidebar. Then, open the Groups menu and click New group.

You can interact with groups in several ways.

  • Create new items for the group
  • Sort items by one of several attributes
  • Drag items to reorder them within the group
  • Use the Groups menu to edit a group's name and color
