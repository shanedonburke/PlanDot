Creating items

To create an item, click on the New item button found in the toolbar or in the .

You may set the item's title (required), description, and location. The Description field also accepts Markdown formatting!

Items with a specified date will appear in the and the .

You can create reoccuring items as well. When creating an item that occurs on a daily or weekly basis, you can choose the week days on which it should occur.

Note: Selected days are highlighted in gray.

Finally, you may specify the item's groups. To add a group, click the Add group text and begin typing the name of an existing group.
